# NorthernBlocks
## Modern 3D Tetris-Style Android Game

NorthernBlocks is a polished, modern 3D Tetris-style falling-blocks game designed for Android devices. Featuring multiple lighting styles, deep progression systems, original music, and customizable themes.

## 🎮 Core Features

### Gameplay
- **3D Tetris Mechanics**: Classic falling-blocks gameplay with full 3D rendering
- **Multiple Difficulty Modes**: Easy, Normal, and Hard with different speed curves
- **Daily Challenges**: Special modifiers and exclusive rewards
- **Smooth Controls**: Optimized touch controls for Android devices

### Visual Excellence
- **5 Lighting Styles**: Neon, Frost/Ice, Voxel, Aurora, and Metallic High-Contrast
- **3D Materials**: Glass, frosted crystal, neon metals, voxel cubes, polished metal, soft plastic
- **Dynamic Effects**: Particle systems, depth-of-field, animated backgrounds
- **Performance Optimization**: 60 FPS target with battery-saving modes

### Audio Experience
- **Original Soundtrack**: 2-3 upbeat gameplay loops, 1-2 calmer menu themes
- **Dynamic Sound Effects**: Movement, rotation, line clears, achievements
- **Seamless Looping**: Professional audio design with smooth transitions

### Progression & Customization
- **Global XP System**: Persistent progression across all game sessions
- **Achievement System**: Structured rewards for various accomplishments
- **Unlockable Themes**: Multiple 3D environments and visual styles
- **Customizable Settings**: Controls, audio, graphics, accessibility options

### Social Features
- **Leaderboards**: Local and online high score tracking
- **Weekly Resets**: Fresh competition cycles
- **Stats Tracking**: Comprehensive gameplay analytics

## 🛠️ Technical Specifications

- **Platform**: Android (API Level 21+)
- **Engine**: Unity 3D
- **Target Performance**: 60 FPS on mid-range devices
- **Graphics**: OpenGL ES 3.0+ with fallback support
- **Audio**: High-quality compressed audio with dynamic mixing

## 🚀 Getting Started

This project is built with Unity 2022.3 LTS or later. Open the project in Unity and ensure Android build support is installed.

### Build Requirements
- Unity 2022.3 LTS+
- Android SDK (API Level 21+)
- Gradle build system
- 4GB+ RAM for development

### Performance Targets
- 60 FPS on devices with Snapdragon 660+ or equivalent
- 30 FPS minimum on older devices with performance mode
- <100MB storage footprint
- <2GB RAM usage during gameplay
